AIBurn Calculator - Project Handoff (Updated)

Project Overview

AIBurn is a React-based AI Token Cost Calculator that helps developers and teams compare AI model pricing and estimate potential savings by switching between OpenAI, Anthropic, Groq, and DeepSeek models.

Location: /Users/bkerwood/projects/aiburn-cost-calculator/aiburn-website

React: 19.2.0
Vite: 7.2.2
Tailwind CSS: Via CDN in index.html
lucide-react: 0.394.0 (note: uses --legacy-peer-deps for React 19 compatibility)
Node: Pure JavaScript (.jsx), no TypeScript
