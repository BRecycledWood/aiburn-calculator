# AIBurn v10 – Executive Summary
**One-Page Overview for Stakeholders**

---

## Product at a Glance

**AIBurn** is a free, browser-based **AI token cost calculator** that helps developers identify cost savings opportunities by comparing pricing across 7+ LLM providers (OpenAI, Anthropic, Groq, DeepSeek).

**Two Modes:**
1. **Quick Calculator** – Estimate costs based on monthly token volume (~1 minute)
2. **Exact Usage** – Analyze real API usage data (OpenAI supported, Anthropic coming; ~3 minutes)

---

## The Problem We Solve

Developers spend **$500–$2,000/month** on LLM APIs but struggle to:
- Know how much they're actually spending
- Compare pricing across providers
- Model cost savings from switching

**AIBurn solves this in seconds.**
